## 🤖 [MyClaw.ai](https://myclaw.ai) が提供

**[MyClaw.ai](https://myclaw.ai)** は専用サーバー上でAIエージェントを提供するプラットフォームです。OpenClaw Master Skills はエコシステム全体から厳選した最高のスキルを毎週更新するコレクションです。

> 🌐 **MyClaw.ai を試す**: [https://myclaw.ai](https://myclaw.ai)

## 🚀 インストール

```bash
clawhub install openclaw-master-skills
# または
git clone https://github.com/LeoYeAI/openclaw-master-skills.git
cp -r openclaw-master-skills/skills/<skill-name> ~/.openclaw/workspace/skills/
```

## 📬 Skillを提出

[Issueを開く](../../issues/new?template=submit-skill.md) か、`skills/` 配下にあなたのskillフォルダを含むPull Requestを送ってください。
