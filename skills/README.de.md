## 🤖 Unterstützt von [MyClaw.ai](https://myclaw.ai)

**[MyClaw.ai](https://myclaw.ai)** ist eine KI-Assistenzplattform mit dedizierten Servern für jeden Nutzer. OpenClaw Master Skills ist unsere kuratierte, wöchentlich aktualisierte Sammlung der besten Skills aus dem Ökosystem.

> 🌐 **MyClaw.ai ausprobieren**: [https://myclaw.ai](https://myclaw.ai)

## 🚀 Installation

```bash
clawhub install openclaw-master-skills
# oder
git clone https://github.com/LeoYeAI/openclaw-master-skills.git
cp -r openclaw-master-skills/skills/<skill-name> ~/.openclaw/workspace/skills/
```
