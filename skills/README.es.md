## 🤖 Impulsado por [MyClaw.ai](https://myclaw.ai)

**[MyClaw.ai](https://myclaw.ai)** es una plataforma de asistente IA con servidores dedicados para cada usuario. OpenClaw Master Skills es nuestra colección curada, actualizada semanalmente, con las mejores skills del ecosistema.

> 🌐 **Prueba MyClaw.ai**: [https://myclaw.ai](https://myclaw.ai)

## 🚀 Instalación

```bash
clawhub install openclaw-master-skills
# o
git clone https://github.com/LeoYeAI/openclaw-master-skills.git
cp -r openclaw-master-skills/skills/<skill-name> ~/.openclaw/workspace/skills/
```
