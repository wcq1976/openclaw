# AI Pet Portal - 设计规范

## 1. 概念与愿景

打造一个专业级 AI 宠物艺术照生成平台，让每只宠物都能成为艺术明星。整体风格现代、温暖、有亲和力，以紫色渐变为主色调，传递科技与温度并存的感觉。用户无需注册即可使用，三步完成艺术照生成。

## 2. 设计语言

### 色彩系统
- **主色**: `#8B5CF6` (紫色)
- **次要色**: `#A78BFA` (浅紫)
- **强调色**: `#F59E0B` (琥珀金)
- **背景色**: `#FAF5FF` (淡紫白) / `#FFFFFF`
- **文字色**: `#1F2937` (深灰) / `#6B7280` (中灰)
- **渐变**: `linear-gradient(135deg, #8B5CF6 0%, #A78BFA 50%, #F59E0B 100%)`

### 字体
- 标题: "Poppins", sans-serif (700/600 weight)
- 正文: "Inter", sans-serif (400/500 weight)
- 中文: "Noto Sans SC", sans-serif

### 间距系统
- 基础单位: 8px
- section padding: 80px 0
- card padding: 24px
- gap: 24px

### 动效
- 所有过渡: 300ms ease
- hover scale: 1.02
- 上浮效果: translateY(-4px)
- loading 动画: 脉冲 + 旋转组合

## 5. 技术方案

### 前端
- 单文件 index.html (HTML + CSS + JS)
- 无框架，纯原生实现
- CSS 变量管理主题
- IntersectionObserver 实现滚动动画

### 后端 API (Cloudflare Workers)
- `/api/generate` - 接收图片 + 风格，返回生成结果
- 使用 OpenAI DALL-E 3 或通义万相 API
- 图片 Base64 传输
- CORS 支持

### 部署
- Cloudflare Pages (前端静态文件)
- Cloudflare Workers (API)
- 预览域名部署测试
